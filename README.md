# Storage-Util

## Ключевая идея: 

Сделать утилиту для командной строки, при помощи которой можно быстро сохранить мысль, которая пришла тебе в голову, не отвлекаясь от работы и даже не выходя из терминала.

## Функционал: 

На начальном этапе необходимо обеспечить возможность создания новой записи, чтения всех созданных записей и их отправки клиенту

## Структура проекта:

Http-клиент - это cli-утилита, которая позволяет пользователю отправить некоторую запись в хранилище, не выходя из терминала

Сервер принимает входящие запросы, парсит их и сохраняет в файловой системе в отведенном для этого json-файле. Вероятно, будет реализован функционал дополнительного шифрования записи/установка ttl
